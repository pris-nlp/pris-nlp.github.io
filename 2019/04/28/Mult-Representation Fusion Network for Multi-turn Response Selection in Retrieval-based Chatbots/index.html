Multi-Representation Fusion Network for Multi-turn Response Selection in Retrieval-based Chatbots

2019-04-28 blog PV:

Introduction

方向：检索式多轮对话的回复选择

从Multi-view引入交互，到SMN完全基于交互，再到DAM多层交互，在检索式对话回复选择任务中基于交互的思想越来越有效，且交互的粒度越多越work。

在基于检索的多轮聊天机器人的回复选择中，作者考虑 多种类型表示 的context-
response匹配。这些表示对context和response在word、n-gram和utterance子序列等不同粒度信息上进行语义编码，并捕获单词之间的短期和长期依赖关系。有了这么多的表示，如何在深度神经网络中融合它们以进行匹配以及它们如何有助于匹配成为了一个研究点。为此，作者提出了一种多表示融合网络（以下简称MRFN），其中这些表示可以在
早期阶段，中间阶段或最后阶段
融合到匹配中。作者在两个数据集上比较了不同的表示和融合策略，实验结果表明，后期融合总是优于早期融合，并且通过在最后阶段融合表示，这篇论文提出的模型明显优于现有方法，并在两个数据集上实现了SOTA。通过消融实验，证明了每种表示对于匹配的影响，这也说明了如何在实际系​​统中选择它们。

本文的贡献:（1）提出了融合多种表示类型的MRFN模型; （2）提出了三种表示的融合策略; （3）在数据集上实现了SOTA;
（4）通过实验，将不同类型的表示对于匹配的贡献做了直观的解释。

Symbols

$\displaystyle U_{i}^{k}$ ：第i个utterance的第k个类型的表示， $\displaystyle k \in {1,
…, K}$ ；

$\displaystyle e^{k}{i,j}、e^{k}{r,j}$ ：第i个utterance和response的第k个类型表示的第j个词；

$\displaystyle U_{i}^{}$ 和 $\displaystyle R^{}$ ：K种表示信息的拼接结果；

$\displaystyle \hat{e}{i,j}、\hat{e}{r,j}$ ： $\displaystyle U_{i}^{}$ 和
$\displaystyle R^{}$ 的第j行，即第k个词；

Model

  1. Representations
  • Word Representations 编码每个word的语义信息

· Character-based Word Embedding 使用字符级编码加CNN减弱OOV问题

· Word2Vec 使用常规的word embedding

  • Contextual Representations 捕捉单词之间的短期依赖

· Sequential Representation 在词嵌入基础上使用GRU获取句子中子序列信息

· Local Representation 在字符嵌入基础上使用CNN获取句子中的n-gram信息

  • Attention-based Representations 编码单词的重要性和单词之间的长期依赖性

· Multi-head Attention 使用多个self-attention的融合来获取不同子空间上的信息

· Cross Attention 使用每个utterance和response之间的交互信息

Experiment

1. 对比实验

和之前存在的一些模型对比 ， 可以看出 FLS 效果比 DAM 提升比较显著； FIS 和 DAM
相差无几，在Douban数据集上略微好一点； FES 效果较差。融合策略效果上： $\displaystyle MRFN_{FES}

<MRFN_{FIS}<MRFN_{FLS}$

2. 消融实验

可以发现每一种类型的表示都很有用，但是去掉其中一个只会带来轻微的变化，这可能是由于不同类型 表示信息的冗余和重叠 ；self-
attention表示的效果在两个中英文数据集上最好，character
embedding在英文数据集上效果好；Contextual和Attention比Word表示信息效果好，Contextual表示信息在模型上表现最好。

3. 模型拓展性

把融合的三种策略移植到到 SMN 模型上也得到了不错的效果，尤其是FLS策略。

4. 其他

作者还实验探究了各个类型表示在不同对话轮数、不同对话长度上的作用占比以及变化。

Conclusion

  1. 在原来SMN、DAM模型两种表示粒度的基础上提出了三种粒度6种表示；
  2. 提出了多表示匹配-聚合(Matching-Aggregation)的三种融合策略；
  3. 验证了各种表示的作用，并验证了context轮次、平均对话长度变化时各个表示的作用情况；
  4. 提出的多表示匹配-聚合策略可推广到其他模型，并在SMN模型上进行实验；
  5. 比DAM模型快1.9x的训练速度，1.7x的测试速度。
  6. 深层的contextual representations和堆叠的attention-based representations未来值得研究。
